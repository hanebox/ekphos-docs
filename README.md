# ekphos-docs

Documentation website for [ekphos](https://github.com/hanebox/ekphos) - a lightweight, fast, terminal-based markdown research tool built with Rust.

**Live:** [docs.ekphos.xyz](https://docs.ekphos.xyz)

## Development

```bash
npm install
npm run dev
```

## Contributing

1. Fork the repo
2. Create a branch (`git checkout -b fix/typo`)
3. Edit docs in `content/docs/*.mdx`
4. Test locally with `npm run dev`
5. Submit a PR

Docs use [MDX](https://mdxjs.com) with [Fumadocs components](https://fumadocs.dev/docs/ui/components).

## Built with

- [Fumadocs](https://fumadocs.dev) - Next.js documentation framework
- [Next.js](https://nextjs.org)
